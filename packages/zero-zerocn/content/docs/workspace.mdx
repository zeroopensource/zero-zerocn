## Usage

Install shadcn cli

```bash
pnpm --filter <workspace> add shadcn@latest -D
```

```json
{
  "scripts": {
    "shadcn": "shadcn"
  }
}
```

```bash
pnpm --filter <workspace> shadcn add https://zerocn.zeroopensource.org/r/<item>.json
```
