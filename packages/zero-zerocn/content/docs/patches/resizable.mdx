Patched Resizable Component.

### Usage

Install package

```bash
pnpm dlx shadcn@latest add https://zerocn.zeroopensource.org/r/resizable.json
```

Install package ([workspace](/docs/workspace))

```bash
pnpm --filter=$WORKSPACE shadcn add https://zerocn.zeroopensource.org/r/resizable.json
```
