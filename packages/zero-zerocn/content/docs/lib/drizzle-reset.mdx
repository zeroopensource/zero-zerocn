### Usage


Install package

```bash
pnpm dlx shadcn@latest add https://zerocn.zeroopensource.org/r/drizzle-reset.json
```

Install package ([workspace](/docs/workspace))

```bash
pnpm --filter=$WORKSPACE shadcn add https://zerocn.zeroopensource.org/r/drizzle-reset.json
```

Add script

```json
{
  "scripts": {
     "db:reset": "pnpm drizzle-kit push --force --config=./src/shadcn/lib/drizzle-reset.config.ts && pnpm drizzle-kit push",
  }
}

```
