### Usage

Install package

```bash
pnpm add @zeroopensource/zero-id
pnpm add @zeroopensource/zero-id --filter <package-name>
npm install @zeroopensource/zero-id
npm install @zeroopensource/zero-id -w <workspace-name>
yarn add @zeroopensource/zero-id
yarn workspace <workspace-name> add @zeroopensource/zero-id
```

Import and use

```js
import { generateZeroId } from '@zeroopensource/zero-id'
const id = generateZeroId()
```
